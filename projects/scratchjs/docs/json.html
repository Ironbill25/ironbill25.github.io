JSON Documentation

JSON (JavaScript Object Notation) is a way to store and organize data. ScratchJS provides blocks to work with Arrays and Objects.

Arrays

Arrays are ordered lists of items. Think of them like a list of grocery items.

You can create arrays using the "create array" block, add items to the start or end, remove items from any position, and get items by their position number. Arrays start counting from 0, so the first item is at position 0.

Create array: apple, banana, orange
Get item at position 0 = apple
Add grape to end = apple, banana, orange, grape
Remove item at position 1 = apple, orange, grape

The Arrays category also has blocks to get length, check if it contains something, join items into text, and sort them.

Objects

Objects are collections of key-value pairs. Think of them like a dictionary where you look up a word (key) to find its definition (value).

You can create objects using the "create object" block, get values using keys, set or change values, and work with keys. Objects can also be converted to text and back again for storage.

Create object: name: John, age: 15
Get value of "name" = John
Set "age" to 16 = name: John, age: 16
Get all keys = name, age
Remember: Arrays use numbers (indexes) to access items, while objects use text (keys) to access values.
